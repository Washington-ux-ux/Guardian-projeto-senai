Sobre o Guardian

Projeto Acadêmico

O Guardian Portal é um projeto desenvolvido como parte do curso técnico do SENAI. Esta aplicação foi criada para demonstrar habilidades em desenvolvimento web, backend e gestão de dados.

Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias modernas como HTML5, CSS3, JavaScript, Node.js com Express, e TypeScript. A arquitetura segue princípios de RESTful API para comunicação entre frontend e backend.

Aviso Importante

Este não é um sistema real. O Guardian Portal é uma simulação educacional criada exclusivamente para fins de aprendizado. Todos os dados, casos e informações apresentados são fictícios e não representam pessoas ou situações reais.

Objetivos de Aprendizagem

Este projeto tem como objetivo principal praticar e demonstrar competências em:

  • Desenvolvimento de interfaces web responsivas
  • Criação de APIs RESTful
  • Gestão de estado e persistência de dados
  • Integração entre frontend e backend
  • Boas práticas de desenvolvimento de software

Sobre o SENAI
